# ContractLens Mobile

AI-Powered Contract Analysis & Legal Intelligence Mobile App built with Flutter.

## Features

- **Contract Analyzer** - Comprehensive AI contract analysis
- **Risk Scorer** - Multi-dimensional risk assessment (financial, legal, operational, compliance)
- **Clause Explainer** - Legal jargon to plain language translation
- **Contract Comparison** - Side-by-side contract diff and analysis
- **Material 3 Dark Theme** - Red/rose legal-themed UI
- **8 Premium Screens** - Splash, onboarding, home, analyzer, risk scorer, clause explainer, comparison, settings

## AI Backend

- **Endpoint**: `sai.sharedllm.com/v1/chat/completions`
- **Model**: `gpt-oss:120b`
- **No API key required**

## Getting Started

```bash
flutter pub get
flutter run
```

## Tech Stack

- Flutter 3.x with Material 3
- Dark theme with red/rose accent
- HTTP for AI API calls
- Provider for state management
